# adapt-authoring-adaptframework

The interface between the authoring tool and the [Adapt framework](https://github.com/adaptlearning/adapt_framework). Handles importing, building, previewing and publishing courses by driving a local copy of the framework files.

Extends `AbstractModule` from [adapt-authoring-core](../adapt-authoring-core); consumes `adapt-cli` at runtime to perform framework operations.
